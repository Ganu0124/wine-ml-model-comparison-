# Comparative Analysis of Machine Learning Algorithms Using Wine Dataset

## Project Overview
This project compares multiple machine learning algorithms using the Wine Dataset from the UCI repository.

## Algorithms Used
- Logistic Regression
- KNN
- Decision Tree
- Naive Bayes
- SVM
- Random Forest
- AdaBoost
- Gradient Boosting
- Extra Trees
- Neural Network
- Perceptron

## Techniques Used
- Feature Scaling
- PCA
- Clustering
- Confusion Matrix
- Feature Importance
- Data Visualization

## Best Model
Random Forest and Extra Trees achieved 100% accuracy.

## Libraries
- scikit-learn
- pandas
- numpy
- matplotlib
- seaborn
- ucimlrepo
